# FPGA-Based-4-Storey-Elevator-Controller

## Description
FPGA-based 4-storey elevator controller developed in Verilog HDL using a Finite State Machine (FSM). Supports internal and external floor requests, automatic door control, directional movement, LED-based motion indicators, and 7-segment floor display on an FPGA.

## Features

- 4-storey elevator control
- Finite State Machine (FSM) implementation
- Internal and external floor request handling
- Automatic door opening and closing
- Up and down movement logic
- LED animation for elevator movement
- 7-segment floor display
- Emergency reset support
- Developed using Verilog HDL

## Hardware Used

- Altera Cyclone II FPGA
- EP2C35F672C6 Development Board
